# Nginx Static Site Deployment

## Project Overview

This project demonstrates how to set up a Linux server and deploy a static website using Nginx and rsync.

## Features

* Linux server setup
* SSH access configuration
* Nginx installation and configuration
* Static website hosting
* Deployment using rsync
* Version control using Git and GitHub

## Technologies Used

* Linux
* Nginx
* HTML
* CSS
* SSH
* rsync
* Git
* GitHub

## Deployment Steps

### Clone Repository

```bash
git clone <repository-url>
```

### Deploy Website

```bash
./deploy.sh
```

### Access Website

Open:

```text
http://SERVER_IP
```
